	tabsize=2,                                      % 制表符（tab）对应的空格数（1 个 tab = 2 个空格）
	frame=single                                    % 代码块边框样式：single（单线框），也可用 none（无框）或 shadowbox（阴影框）

  \section{测试概述}
   本测试报告涵盖了高铁订票管理系统的功能测试、性能测试和用户界面测试。测试旨在确保系统的各项功能符合需求规格说明书，并在不同环境下表现出良好的性能和用户体验。
    \subsection{测试内容}
    \begin{itemize}
      \item 功能测试：验证系统的各项功能是否按预期工作，包括用户注册、登录、车票查询、订票、退票等功能。
      \item 性能测试：评估系统在高并发情况下的响应时间和稳定性。
      \item 用户界面测试：检查界面的易用性和一致性，确保用户操作流畅。
    \end{itemize}
    \subsection{测试环境}
    测试在Windows 10操作系统上进行，使用Qt 6作为开发环境。
